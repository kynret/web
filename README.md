# Projekt Zaliczeniowy Klon Netflix

## Użyte technologie

- HTML5 / CSS3
- Sass (SCSS) z podziałem na pliki modułowe i zmienne
- Środowisko uruchomieniowe Vite
- Menedżer pakietów PNPM
- Font: Google Fonts (Inter)

## Podejście projektowe

Projekt został wykonany w metodologii **Mobile First**. Główny (breakpoint) został ustawiony na szerokości `768px`, adaptując interfejs do ekranów desktopowych.
